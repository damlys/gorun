# Files extensions

Go:

- `example.*.gotpl` - Go template files
- `example.gotpl.*` - files generated with Go templates

Terraform:

- `example.*.tftpl` - Terraform template files
- `example.tftpl.*` - files generated with Terraform templates

SOPS:

- `example.dec.*` - SOPS decrypted files
- `example.enc.*` - SOPS encrypted files
- `example.secret.enc.yaml` - Kubernetes secret manifests encrypted with SOPS
